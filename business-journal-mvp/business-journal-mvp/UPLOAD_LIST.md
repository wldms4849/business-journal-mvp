## 💡 쉬운 선택 방법

### Windows:
1. 폴더 열기
2. 첫 번째 파일 클릭
3. **Shift + 클릭** 마지막 항목 → 전체 선택됨
4. .git 폴더만 **Ctrl + 클릭**으로 선택 해제

### Mac:
1. 폴더 열기
2. **Cmd + A** 전체 선택
3. .git 폴더만 **Cmd + 클릭**으로 선택 해제

## 🎬 실제 업로드 순서

```
1. GitHub 페이지 열기
   https://github.com/wldms4849/business-journal-mvp

2. "Add file" → "Upload files" 클릭

3. 파일 선택
   - Ctrl/Cmd + A로 전체 선택
   - 또는 마우스로 드래그해서 전체 선택

4. GitHub 창으로 드래그 앤 드롭
   (파일들이 업로드되는 게 보임)

5. 커밋 메시지 입력
   "Initial commit: 사업일기 MVP 완성"

6. "Commit changes" 버튼 클릭

7. 완료! 🎉
```
